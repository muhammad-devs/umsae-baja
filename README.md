# Electrical & Instrumentation Section

## Overview
This repository serves as the central codebase for the baja vehicle's software systems. Which currently covers data acquisition and the 4WD/2WD drivetrain actuation.

## Core Systems
1. Drivetrain Actuator Control
- Controls the shift mechanism between 4WD and 2WD using a linear actuator
- The internal hall effect sensor of the actuator counts pulses to track position, which is translated in code to inches of travel — ensuring exactly 0.5 inches of stroke for precise gear engagement

2. DAQ Data Logger
- Two hall effect sensors handle speed measurement: one counts pulses from the magnetic field generated by the engine's spark plug to calculate primary RPM, and a second mounted on the axle captures secondary CVT speed
- CVT belt temperature is read via an MLX90614 IR sensor
- Primary RPM, secondary speed, and belt temperature are displayed live on an LCD in conjunction with a high-temp warning LED for the CVT belt
- All data is logged to a FAT32 formatted SD card as a .CSV file, giving the powertrain, and gearbox sections actionable data to improve their designs

## Hardware
1. Arduino (Elegoo) Uno (Microcontroller for the DAQ system)
2. Arduino (Elegoo) Nano (Microcontroller for the e-shifter)
3. Progressive Automations Linear Actuator (Model: PA-01-1-56-HS-12VDC)
4. H-bridge Motor Driver (Model: DRV8874)
5. Single and Dual Zone IR Sensor (Model: MLX90614)
6. Hall Effect Sensor (Model: A3144) x2
7. LCD display (Model: CFAH1602D-TMI-ET)
8. MicroSD card Breakout Board
